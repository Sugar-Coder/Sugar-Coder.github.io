## 知识补充
### shell
sed的 ``\1-9`` 提取匹配的模式。s动作表示替换 ``s'/oldstr/newstr/g``
```sh
$ echo QEMU emulator version 7.2.0 | sed 's/QEMU emulator \([a-zA-Z]*\) \([0-9]\)\.\([0-9]\).*/\1.\2.\3/'
version.7.2
```
